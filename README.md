Sistema de gestión de apuestas hípicas desarrollado en Java como proyecto académico para la materia Diseño de Aplicaciones.
Permite:
Gestión de jornadas y carreras.
Registro de caballos participantes.
Apuestas con distintas modalidades.
Confirmación y liquidación de apuestas.
Gestión administrativa de carreras.
Actualización en tiempo real mediante Observer y Server-Sent Events (SSE).
Tecnologías: JavaSpring Boot, MVP, HTML, CSS, JavaScript, Observer Pattern,SSE
